# Banco de Dados

Este diretório contém os scripts SQL usados para criar e popular o banco de dados do sistema de cadastro de alunos.

## Como usar

1. Abra o SQL Server Management Studio (ou outra ferramenta).
2. Execute o arquivo `script_criacao.sql` para criar o banco e as tabelas.
3. (Opcional) Execute `script_insercao.sql` para adicionar dados de exemplo.
